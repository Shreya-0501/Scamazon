This folder contains raw python code before being made compatable with the flask adapter

This database consists of 4 tables-
  -Buyer
  -Vendor
  -Items
  -Orders

The vendor table consists of login info for the vendor
The buyer table consists of login info for the buyer

The item table consists of information about the item was ordered by whom and when
The order table conisists of information about which item was ordered by whom and when
